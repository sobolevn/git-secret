git-secret-usage - prints all the available commands.
=====================================================

## SYNOPSIS

    git secret usage


## DESCRIPTION
`git-secret-usage` is used to print all the available commands.


## OPTIONS

    -h  - shows this help.


## MANUAL

Run `man git-secret-usage` to see this note.
