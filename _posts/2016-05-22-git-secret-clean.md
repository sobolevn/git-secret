git-secret-clean - removes all the hidden files.
================================================

## SYNOPSIS

    git secret clean [-v]


## DESCRIPTION
`git-secret-clean` deletes all the encrypted files. This command can produce a verbose output, printing which files are deleted.


## OPTIONS

    -v  - shows which files are deleted.
    -h  - shows this help.
