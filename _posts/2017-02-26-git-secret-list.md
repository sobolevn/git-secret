git-secret-list - prints all the added files.
=============================================

## SYNOPSIS

    git secret list


## DESCRIPTION
`git-secret-list` prints all the currently added tracked files from the `.gitsecret/paths/mapping.cfg`.


## OPTIONS

    -h  - shows this help.


## MANUAL

Run `man git-secret-list` to see this note.
